ABOUT THE PROJECT

Abstract

Blockchains LLC. is a company dedicated to improving the world through revolutionary block chain technology, situated on about 67,000 acres of land in northern Nevada. In the process of purchasing that land for their business, they also inherited the wild horses that live on that land. Team #02’s project is to ensure that the wild horses can live safe, healthy lives through the use of the team’s project to identify and keep up to date vaccination records for wild horses.

Main Goals and Objectives

The primary goal of Team #02 is to create a mobile application that is capable of searching for wild horses based on their physical attributes.

Main Functionality and Characteristics

The application will be loaded through an iOS device. The user will be able to search through horses by name, location, or features as a main functionality. Additionally the user will be able to log in and do certain actions based on their status, such as generate reports or edit horse records. When pulling up a horse record the user will be able to see information on the horse as well as its band members.

Intended Users

The intended users of this app are darters working with the American Wild Horse Society and Blockchain LLC to administer medicine and birth control to wild horses. This audience includes field darters, herd leaders, and admin profiles that all have different user needs. This app may later be expanded to be used by other groups seeking to monitor and cater to the needs of wild horse populations.

Technology Description

The main technology that will be used to create the application is Apple’s Swift and XCode. Working together with Blockchain LLC the team will create an API to allow it to interface with the data for the horses.

Dependability

This product has several key components that ensure the reliability, security and safety. To ensure reliability in the wilderness, the app will be designed to work largely without internet or cellular service. Security will be taken care of by the login software in the app. Only registered admins will be able to perform admin level actions, herd leaders will be the only users able to approve changes to their herd, and regular users will be limited in what they can do and access. Finally, the app will be safe by keeping complete control in the hands of the users. Instead of making a complete guess on which horse the user is searching for, the user will be able to look through bands and make an informed decision on which horse is which, minimizing possible damage to horses.
